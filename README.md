# Hotel Room Booking System

This is a simple console-based project written in C that demonstrates how a basic hotel room booking system works.  
The program allows the user to view different room types, check available facilities, enter booking details, and calculate the total cost of stay.

## What the program does
- Shows available room types such as Standard, Deluxe, and Luxury
- Displays facilities provided with each type of room
- Takes user details like name, contact number, and number of rooms
- Calculates the total bill based on days of stay and rooms booked
- Checks room availability before confirming the booking

- C programming language
